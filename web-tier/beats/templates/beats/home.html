How is SmartBeats Utilized?

SmartBeats is a software application that supports three functions: the development and adjustment of deployment areas, the annual shift bidding process and daily shift scheduling. To use the application, a user will be able to add a data source and choose the type of beats that they would like to be generated.These options are as follows:

  • Priority of Calls for service
  • Specifiy number of beats to create
  • Specify number of Calls for service per beat
  • Show only Calls for service that converted to incidents

The user will then be tasked to generate the Glendale map after selecting one of the options as stated above.
