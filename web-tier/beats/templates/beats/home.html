How is SmartBeats Utilized?

Smartbeats is a software application that supports the annual shift bidding process and daily shift scheduling by providing tools that filter incident data and generate patrol areas within a city.

The user can either choose 'New Beats Generation' to upload a dataset or select 'Utilize Existing Data' to reuse the last uploaded data source to create patrol areas.
